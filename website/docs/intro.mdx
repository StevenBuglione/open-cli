# Docs Hub

`open-cli` ships two binaries — **`ocli`** (the client) and **`open-cli-toolbox`** (the hosted runtime) — that together turn OpenAPI descriptions into a remote-only, policy-aware command surface.

**New here?** Go to [Installation](./getting-started/installation) → [Quickstart](./getting-started/quickstart). You can have a generated command tree running in under 5 minutes.
