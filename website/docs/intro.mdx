# Docs Hub

`oas-cli-go` ships two binaries — **`oascli`** (the CLI) and **`oasclird`** (the runtime daemon) — that together turn OpenAPI descriptions into a local, policy-aware command surface.

**New here?** Go to [Installation](./getting-started/installation) → [Quickstart](./getting-started/quickstart). You can have a generated command tree running in under 5 minutes.
